## Commands

in 'template-flask' folder

Create venv folder 
```
python3 -m venv venv 
```

Activate the virtual environnement
```
source venv/bin/activate 
```

install required python modules
```
pip3 install -r requirements.txt  
```

Launch 
```
flask run
```

deactivate the virtual environnement
```
deactivate 
```
